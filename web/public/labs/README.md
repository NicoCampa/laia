Place model lab icons in this folder.

The dashboard looks for these filenames first:

- `qwen.svg`, `qwen.png`, `qwen.jpg`, or `qwen.webp`
- `google.svg`, `google.png`, `google.jpg`, or `google.webp`
- `meta.svg`, `mistral.svg`, `deepseek.svg`, `microsoft.svg`, `ibm.svg`
- `openai.svg`, `cohere.svg`
- `bonsai.svg` for Prism ML's Bonsai models

Unknown model families fall back to a generated initials mark.
